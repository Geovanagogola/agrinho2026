# agrinho2026# ☀️ AgroSolar

Projeto educativo sobre a utilização da energia solar para otimizar a produção agrícola.

## Objetivos

- Demonstrar os benefícios da energia solar no campo.
- Explicar onde encontrar placas solares.
- Mostrar quem pode auxiliar na implantação.
- Apresentar formas de obter informações e capacitação.
- Simular a economia gerada pelo sistema.

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Como Executar

1. Baixe os arquivos.
2. Coloque todos na mesma pasta.
3. Abra o arquivo index.html.
